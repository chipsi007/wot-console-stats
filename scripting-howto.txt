Babel:

Install locally:
  npm install --save-dev babel-cli

Instal react preset:
  npm install --save-dev babel-cli babel-preset-react

Create babelrc:
  echo '{ "presets": ["react"] }' > .babelrc

Compile JSX:
  ./node_modules/.bin/babel static/app.jsx --out-file static/app.js

Complile JSX and watch:
  ./node_modules/.bin/babel static/app.jsx --watch --out-file static/app.js



SASS:

Install (macOS):
  sudo gem install sass

Compile SASS minified, without sourcemaps:
  sass --sourcemap=none main.sass main.css --style compressed
